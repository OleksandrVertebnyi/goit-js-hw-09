## 📁 Вміст проєкту:

- `src/js/1-gallery.js` — скрипт для галереї зображень з бібліотекою `SimpleLightbox`
- `src/js/2-form.js` — скрипт для збереження даних форми у `localStorage`
- `src/1-gallery.html` та `2-form.html` — HTML-файли сторінок
- `vite.config.js` — налаштування Vite для збірки проєкту

## ⚙️ Збірка:
Проєкт зібрано з використанням **Vite**. Базовий шлях налаштовано: `base: '/goit-js-hw-09/'`.
